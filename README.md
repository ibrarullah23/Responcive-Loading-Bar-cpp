# Responsive Progress Bar in C++

This project showcases a **responsive progress bar** implemented in C++.

## Demo
![Demo](demo.gif)  

## Usage
1. Compile the `Loading bar.cpp` file:
    ```bash
    g++ "Loading bar.cpp" -o progressbar
    ```

2. Run the executable:
    ```bash
    progressbar.exe
    ```
